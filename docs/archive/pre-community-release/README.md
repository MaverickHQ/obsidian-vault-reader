# Pre-Community-Release Archive

This folder preserves historical planning material from the earlier, broader project scope.

The active public release is now an Obsidian community plugin focused on local speed reading, reader controls, settings, and optional in-note highlight. Device export, SD-card workflows, BYOK AI, paid-readiness planning, and selection-only reading are deferred until they re-enter through the active requirements, design, ADR, task list, and TDD gates.

Archived files are kept for context only. They are not release commitments.
